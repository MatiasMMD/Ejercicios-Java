## ▶️ Cómo compilar y ejecutar
Desde la carpeta `src`:

```bash
# Compilar un archivo específico
javac -d ../bin Ej5.java

# Ejecutar el archivo compilado
java -cp ../bin Ej5

# Compilar todos los ejercicios de golpe
javac -d ../bin *.java

```
